# Architecture Decision Records (ADRs)

Place any architecture decisions here as separate markdown files.

These are decisions that have ALREADY been made and agents must respect.
If no decisions exist yet, the architect agent will propose them in Phase 2.

## Format

Each ADR should follow this structure:

```markdown
# ADR-001: [Title]

## Status: accepted | proposed | deprecated

## Context
What is the situation that requires a decision?

## Decision
What was decided?

## Consequences
What are the trade-offs?
```
